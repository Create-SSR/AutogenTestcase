# AutoProductTestCase
借助AI大模型帮助生成测试用例

### 编程语言：python
### 相关python库及技术点：
1,前端：streamlit库  
2,openai：集成deepseek模型，后续可增加其他ai大模型；
3,autogen框架 构建智能体交互应用  
4,xlsxwriter库，支持生成xls文件  

### 预计实现功能：
1，可上传文件输入需求描述  
2，可手动输入需求描述  
3，支持配置用例分类，如功能用例、兼容用例、异常用例等  
4，支持配置生成用例数量  
5，下载生成测试用例：支持md和xlsx文件格式
